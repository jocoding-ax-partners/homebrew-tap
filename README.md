# homebrew-tap

> Homebrew formulas for [axhub](https://cli.jocodingax.ai) by Jocoding AX Partners.

## Install

```bash
brew install jocoding-ax-partners/tap/axhub
```

Or manually:

```bash
brew tap jocoding-ax-partners/tap
brew install axhub
```

## Formulas

- `axhub` — CLI for the axhub developer platform.

## Status

**v0.0.x** — placeholder formula. Real URLs/checksums are written automatically by
`goreleaser` on each upstream release tag.
